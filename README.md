# BuildYourHouse

## Description

This is a simple project to build a house using ReactNative.

## Installation

### Prerequisites

- [Node.js](https://nodejs.org/en/)

### Clone the repository & Install dependencies

```bash
git clone
```

###

```bash
npm install
```

### IOS

```bash
cd ios
pod install
```

```bash
npx start react-native run-ios
```

### Android

```bash
cd android
./gradlew clean
```

```bash
npx start react-native run-android
```
